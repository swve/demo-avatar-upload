# Avatar Image Upload demo 

## Install 

```bash
npm install

# or

yarn install

# or

pnpm install

# or

bun install
```


## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

## Use 

Click on the choose file button to get started
